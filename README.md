# proxy-me

## Installation

```shell
npm install -g
```

## Usage

```shell
  Options:

    -h, --help             output usage information
    -V, --version          output the version number
    -t, --target [target]  REQUIRED - The prefix you'd like for all paths under source
    -s, --source [source]  REQUIRED - The prefix for all endpoints you'd like to proxy to
    -d, --dir [dir]        Directory of where to serve static content from
    -p, --port [port]      Port that you'd like the application to run on
    -h, --https            Set if you'd like the proxy server to run on https (self-signed cert)
```
